# Flutter Calculator App

## Overview

This project is a simple Flutter-based calculator application. It allows users to input two numbers and perform an addition operation. The result is displayed instantly on the screen.

The app demonstrates basic concepts such as user input handling, state management, and UI building using Flutter.

---

## Features

* Input fields for two numbers
* Numeric keyboard support
* Addition operation
* Real-time result display
* Simple and clean user interface

---

## Technologies Used

* Flutter
* Dart
* Material UI components

---

## Project Structure

* `main.dart`

  * Contains the entire application logic
  * Includes:

    * `MyApp`: Root widget
    * `Calculator`: Main calculator screen

---

## How It Works

1. The app launches with the Calculator screen.
2. User enters:

   * First number
   * Second number
3. On clicking the "Add" button:

   * The input values are converted into integers
   * The sum is calculated
   * The result is displayed on the screen

---

## How to Run

1. Install Flutter on your system.
2. Create a new Flutter project or replace the content of `main.dart` with this code.
3. Run the following command:

   ```
   flutter run
   ```
4. Launch the app on an emulator or physical device.
